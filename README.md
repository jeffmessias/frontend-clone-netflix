# 🚀 Imersão Alura Front-End com IA

Este repositório contém os projetos e exercícios desenvolvidos durante a **Imersão Front-End com Inteligência Artificial da Alura**.

O objetivo da imersão é praticar os **fundamentos do desenvolvimento front-end**, utilizando **HTML, CSS e JavaScript**, além de explorar como a **IA pode auxiliar no processo de criação e desenvolvimento** de interfaces web modernas.

***

## 🧠 Conteúdos abordados

*   Estruturação de páginas com **HTML5**
*   Estilização com **CSS3**
*   Interatividade com **JavaScript**
*   Boas práticas de Front-End
*   Uso de **Inteligência Artificial** como apoio no desenvolvimento
*   Construção de layouts inspirados em aplicações reais

***

## 📁 Estrutura do projeto

A estrutura do repositório pode conter, por exemplo:

    /
    ├── index.html
    ├── styles.css
    ├── script.js
    ├── assets/
    │   └── imagens, ícones, etc.
    └── README.md

> A organização pode variar conforme as aulas e desafios da imersão.

***

## ▶️ Como executar o projeto

1.  Clone este repositório:
    ```bash
    git clone https://github.com/jeffmessias/imersao-alura-frontend-ia.git
    ```

2.  Acesse a pasta do projeto:
    ```bash
    cd imersao-alura-frontend-ia
    ```

3.  Abra o arquivo `index.html` no navegador  
    ou utilize uma extensão como **Live Server** no VS Code para melhor experiência.
